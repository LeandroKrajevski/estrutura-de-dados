Atividades de Estrutura de Dados

Descrição

Este repositório contém a implementação das atividades avaliativas da disciplina de Estrutura de Dados.
Os exercícios abordam conceitos fundamentais da programação em C, como vetores, matrizes, ponteiros, structs, pilhas e filas.

## Estrutura do Projeto

### Atividade 1

Exercícios focados em lógica básica e manipulação de estruturas simples:

1. Inversão da ordem de palavras digitadas pelo usuário
2. Inversão de um vetor de números
3. Leitura de matriz 3x3 e multiplicação dos valores
4. Operações com matriz identidade
5. Multiplicação entre vetor e matriz
6. Cadastro de alunos com separação entre aprovados e reprovados
7. Cadastro e busca de livros por título
8. Uso de ponteiros para manipulação de vetor
9. Ordenação de três números usando ponteiros
10. Identificação do aluno com maior nota usando alocação dinâmica

---

### Atividade 2

Implementação de **pilhas (stack)** usando listas encadeadas:

1. Verificação de expressões válidas (parênteses, colchetes e chaves)
2. Inversão de uma string utilizando pilha

---

### Atividade 3

Implementação de **filas (queue)**:

1. Simulação de atendimento de clientes com cálculo de tempo de espera
2. Sistema de fila com prioridade para impressão de documentos

---

## Explicação da solução

Os exercícios foram desenvolvidos com o objetivo de praticar conceitos importantes de Estrutura de Dados:

  - **Vetores e Matrizes:** utilizados para armazenar e manipular dados de forma estruturada
  - **Structs:** usadas para representar entidades como alunos, livros e clientes
  - **Ponteiros:** aplicados para manipulação de memória e passagem por referência
  - **Alocação dinâmica:** utilizada em problemas mais avançados
  - **Pilhas:** estrutura LIFO aplicada em validação de expressões e inversão de strings
  - **Filas:** estrutura FIFO aplicada em simulações de atendimento e sistemas com prioridade

---

## Conclusão

As atividades permitiram o desenvolvimento da lógica de programação e o entendimento prático das principais estruturas de dados, fundamentais para a área de Engenharia de Software.
